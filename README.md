# 🥔 Potato Bag Weight Calculator

A simple **Streamlit web app** to calculate potato bag weights quickly.

Developed for **Soham Traders** to replace manual notebook calculations and automatically compute bag totals and overall weight.

## Features
- Enter farmer details
- Add up to **1000 bags**
- Enter **10 weights per bag**
- Automatic bag total calculation
- Automatic total weight calculation
- Save farmer records
- Download printable **PDF receipt**
- Mobile-friendly interface

## Run the App

Install requirements:

pip install -r requirements.txt

Run:

streamlit run app.py
